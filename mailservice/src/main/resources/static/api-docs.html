BeOne Mock-Mail-Microservice

API Documentation for the Mailer Service

This is a mockup microservice for development. Use this api doc for sending and recieving mail through the mock microservice. No actual mail will be sent, it's only stored and viewed in this web frontend.

Send Mail

CURL

Sending a request

curl -i -X POST -H "Content-Type:application/json" -d '{"recipients": ["test@test.com"], "subject": "subject", "contents": "content"}' "http://127.0.0.1:8080/api/send"

Expected Response

Content-Type: application/json;charset=UTF-8
Transfer-Encoding: chunked
Date: Thu, 11 Aug 2016 12:19:34 GMT

{"message":"ok","success":true}

Get Mails

CURL

Sending a request

curl -i -X GET -H Accept:application/json "http://127.0.0.1:8080/api/search?q=1&page=0"

Response to expect

Content-Type: application/json;charset=UTF-8
Transfer-Encoding: chunked
Date: Thu, 11 Aug 2016 12:21:45 GMT

[{"id":10,"recipient":"test@test.com","subject":"subject","contents":"content","sentAt":"2016-08-11T16:26:20.587"}}}]
